v2.7.0.0  (June 28, 2018)

Enhancements
  • Users can add comments when sharing charts by email. You can share a chart with your customized comments for all metrics, drilldowns, and analytics.
  • Reorder of columns in grids is as simple as drag and drop. Users can change columns to any order in grids if needed.

Bug Fixes
  • Fixed: Layout changes of a dashboard are not reflected for other users
  • Fixed: TUFF upload messages are more informative about the location of errors
  • Fixed: Grids with numerous columns show a horizontal bar for better layout
  • Fixed: Donut charts support many colors to avoid duplicates
  • Fixed: Regular internal bug fixes

v2.6.0.0  (May 14, 2018)

Enhancements
  • Add Drilldowns & Analytics to dashboard. Any drilldown or metric chart can be added to the dashboard for easier access. The dashboard chart will preserve the chosen chart view, filters, and vertical-axis unit.
  • Alerts help you watch any metric, drilldown, or analytics in the system. You can choose threshold alerts that notify you based on the data variation, or periodic that notify based on recurring time periods.
  • Separation Type has been added to Turnover chart views to see the reasons of termination (voluntary, involuntary, retirement, and others).
  • Overall system performance improvement for faster page loads.

Bug Fixes
  • Fixed: Reports per Manager calculations has been adjusted to count active employees only
  • Fixed: Turnover grids show terminated employees only, not active ones
  • Fixed: Grid columns show well after sorting
  • Fixed: Regular internal bug fixes

v2.5.0.0  (April 16, 2018)

Enhancements
  • Automatic Calculation for Commute Distance. The system uses Google Maps to find the home-office driving distance for all employees
  • Default Chart Views can be set for each drilldown or analytics chart seperately. This configuration is accessible from Site Settings menu
  • Menus items have been reduced to group similar charts under the same category. The new categories are: Recruiting, Financial, Talen, and Organization
  • Smaller Data-Sources boxes for better interface

Bug Fixes
  • Fixed: Overall system-performance improvement
  • Fixed: Resize icons are removed from dashboard for better interface
  • Fixed: System directory permissions
  • Fixed: Regular internal bug fixes

v2.4.0.0  (March 22, 2018)

Enhancements
  • All drilldown and analytics grids can be exported to CSV
  • Custom Fields can be added anywhere in the TUFF file. You no more need to move them to the right of the sheet.
  • "Performance" field has been added to chart views and filters
  • The status of the TUFF upload will appear as a progress bar

Bug Fixes
  • Fixed: Filters are searchable fields by typing in the text boxes
  • Fixed: Overall system-performance optimization
  • Fixed: Some page-load errors have been fixed
  • Fixed: Regular internal bug fixes
