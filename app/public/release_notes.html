v2.3.0.0  (Feb 1, 2018)

Enhancements
  • Dynamic Fields are supported in TUFF uploads. Define the format of your TUFF file by choosing the name of the fields (CSV headers), their order, and their availability (enabled or disabled). This enhancement allows you configure TrenData system to match your file format instead of configuring your file to match the TrenData expected format.
  • Custom Fields are listed in "Manage Columns" popup that controls the visibility of columns in drilldown and analytics grids.

  • Fixed: "Custom" word has been removed from labels in drilldown and analytics filter panel.

  • Fixed: Values in analytics are rounded to the nearest tenth.
  • Fixed: Vertical Axis setting reflects correctly on the chart by showing percentage, value, or dollar amounts.
  • Fixed: Benefit Costs reads from matching month in Financial Inputs when Metric History of that month is chosen.
  • Fixed: Regular internal bug fixes
