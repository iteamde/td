Overview

TrenData's Natural Language Processing (NLP) engine allows you to query your data using plain English sentences. Once you view the result, you can add it to the dashboard for later references.

The engine also provides as-you-type real-time suggestions that help you construct your questions.


Examples

Here are some example queries that can be run on your data:
  • Employees from china born before 1982
  • Male employees with salary over 50k
  • Average of salary per country

Reserved Words

In this section, we list the various saved words that the system understands in each category. For example, these sentences are all understood by the system to have the same meaning:
  • How many employees per department
  • How many employees in each department
  • How many employees grouped by department
  • How many employees by department
  • How many employees for each department
